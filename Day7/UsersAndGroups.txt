## Data explain in cat /etc/passwd
userName:encryptedPasswordStored:userId:groupId:comment:rootDirectoryOfUser:loginShell
root:x:0:0:root:/root:/bin/bash
vagrant:x:1000:1000::/home/vagrant:/bin/bash

## To add a user
useradd userName
useradd devops

add user in existing group
useradd -g groupName userName
useradd -g devops sreuser

modify user group
usermod -g groupName userName
usermod -g devtest sreuser
